# Mappings

- *,dn* toggle line numbers
- *,dR* reloads .vimrc
- *,dov* opens .vimrc 
- *,doV* opens .vimrc in a new tab
- *,dot* opens .tmux.conf in a new tab
- *,doT* opens .tmux.conf in a new tab
- *,dobb* opens .bashrc
- *,doBB* opens .bashrc in a new tab
- *,dobp* opens .bash_profile
- *,doBP* opens .bash_profile in a new tab
